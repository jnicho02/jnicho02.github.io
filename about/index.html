This site

I built my first 'web' site in 1987 using the teletext-like character set of the BBC Micro to publish the Top 40 every Tuesday morning. I listened to the chart rundown just before school and rushed to transcribe it and be read on the computer network by first break.

Since then I have blogged on-and-off whenever I needed to make notes.

This new incarnation is a personal diary system keeping track of books I have read, events I have attended, or even spoken at. The content is written in text files and sotred in github so that I own my personal content. It is written in Markdown text and run through a customised Ruby Middleman static html generator. The resulting html is easy to host on github pages and less hackable than a Content Management System. Layout is made responsive by using Bootstrap 4. This is bleeding edge, so there may by errors.
